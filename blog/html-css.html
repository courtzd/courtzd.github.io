What are the best practices associated with using classes vs. ids?

Firstly, what is an ID?

  • IDs are used to style a single element.
  • IDs are specific and unique.
  • Each element can only have one ID and each page can only have one element with that ID.
  • You can recognize an ID in CSS as it is defined with a # followed by the name.

Ok, so what's a class then?

  • Classes style multiple elements.
  • An element may only be able to have one ID, but it can have multiple classes.
  • Classes are defined by a period ., followed by the class name.
